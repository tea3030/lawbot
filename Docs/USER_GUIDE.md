# 법령 Q&A 챗봇 사용자 가이드

## 개요

법령 Q&A 챗봇은 인허가 절차 및 관련 법령에 대한 질문에 답변을 제공하는 AI 기반 웹 애플리케이션입니다.

## 주요 기능

### 1. 법령 질의응답
- 법령 관련 질문에 대한 AI 답변 제공
- 관련 법령 및 조문 자동 인용
- 법령 출처 정보 표시

### 2. 법령 검색
- 키워드 기반 법령 검색
- 관련도 순 정렬
- 법령 상세 정보 조회

### 3. 대화 관리
- 대화 세션 저장 및 불러오기
- 대화 히스토리 관리
- 대화 목록 조회

### 4. 즐겨찾기
- 중요한 답변을 즐겨찾기에 저장
- 로컬 스토리지에 자동 저장
- 즐겨찾기 목록에서 빠른 접근

### 5. 답변 복사
- 답변 내용을 클립보드로 복사
- 법령 출처 정보 포함

## 사용 방법

### 질문하기

1. 채팅 입력창에 질문을 입력합니다.
2. "전송" 버튼을 클릭하거나 Enter 키를 누릅니다.
3. AI가 관련 법령을 검색하고 답변을 생성합니다.
4. 답변과 함께 관련 법령 출처가 표시됩니다.

### 질문 예시

- "태양광 발전소 건설 시 필요한 인허가는?"
- "건축법 제11조의 건축허가 절차는?"
- "신재생에너지법과 전기사업법의 관계는?"
- "환경영향평가가 필요한 경우는?"

### 법령 검색

1. 헤더의 "법령 검색" 메뉴를 클릭합니다.
2. 검색 키워드를 입력합니다.
3. 검색 결과를 확인하고 원하는 법령을 선택합니다.

### 즐겨찾기 사용

1. 답변 메시지에 마우스를 올리면 하트 아이콘이 나타납니다.
2. 하트 아이콘을 클릭하여 즐겨찾기에 추가합니다.
3. 헤더의 "즐겨찾기" 버튼을 클릭하여 저장된 답변을 확인합니다.
4. 즐겨찾기에서 답변을 선택하여 다시 확인할 수 있습니다.

### 답변 복사

1. 답변 메시지에 마우스를 올리면 복사 아이콘이 나타납니다.
2. 복사 아이콘을 클릭하여 답변을 클립보드에 복사합니다.
3. 복사 성공 시 체크 아이콘이 표시됩니다.

## 주의사항

### 면책 조항

- 이 챗봇의 답변은 참고용이며, 정확한 법적 판단은 전문가와 상담하세요.
- 법령은 수시로 개정될 수 있으므로, 최신 법령 정보를 확인하세요.
- 실제 인허가 절차는 지역별로 다를 수 있습니다.

### 사용 제한

- 질문은 1000자 이하로 입력해주세요.
- 법령 검색 키워드는 100자 이하로 입력해주세요.
- 네트워크 연결이 필요합니다.

## 문제 해결

### 답변이 생성되지 않는 경우

1. 네트워크 연결을 확인하세요.
2. 질문을 다시 입력해보세요.
3. 잠시 후 다시 시도해주세요.

### 오류 메시지가 표시되는 경우

1. 브라우저를 새로고침하세요.
2. 브라우저 캐시를 삭제하세요.
3. 문제가 지속되면 관리자에게 문의하세요.

## 문의

문제가 발생하거나 개선 사항이 있으면 관리자에게 문의해주세요.
